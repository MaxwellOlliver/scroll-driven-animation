# 🎬 Scroll Driven Animation

A React-based project that demonstrates how animations can be controlled by scroll position. As the user scrolls, the animation progresses forward or backward, creating a smooth and interactive experience.

## 🚀 Demo

👉 https://maxwellolliver.github.io/scroll-driven-animation/

## ✨ Features

- 🎯 Scroll-based animation control
- ⏩ Forward and backward animation based on scroll direction
- 🎞️ Smooth transitions tied to scroll progress
- 📱 Responsive behavior

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- CSS / Custom animations
